# 环节执行与评委判准规范（适配Judge Agent与全环节执行）
## 一、各环节核心功能与执行规范
### （一）立论环节（一辩陈词）
1.  核心功能：完整输出己方的定义、标准、核心论点、价值框架，完成基础论证，为全场比赛搭建己方的讨论体系。
2.  执行规范：
    - 结构清晰，严格遵循标准立论结构，逻辑层层递进；
    - 语言简洁，避免过度晦涩的术语与冗余的修饰，保证评委能一次性听懂核心框架；
    - 预判攻防，在结尾明确预判对方的核心攻击点，提前做好防守铺垫；
    - 时长适配，严格控制在赛制规定时长内，核心论点必须充分论证，避免超时导致核心内容未输出。

### （二）质询与首质环节
1.  首质核心功能：确认共识、锁定分歧、框定讨论范围，不追求一次性击穿对方立论。
2.  中段质询核心功能：拆解对方核心论点、攻击对方框架漏洞、推进己方论证，完成核心攻防的爆破。
3.  执行规范：
    - 首质只做框定确认，不做过度攻防，避免提前暴露己方的攻击思路；
    - 质询问题必须闭环，一组问题只针对一个核心目标，避免问题零散、无焦点；
    - 严格控制节奏，避免对方在质询环节长篇大论，浪费己方环节时间。

### （三）申论/小结环节
1.  核心功能：填坑（补充立论未展开的论证）、洗白（澄清对方对己方的曲解）、深化（推进己方论证的深度）、结交锋（总结前序环节的攻防成果）。
2.  执行规范：
    - 严格遵循「定→打→结」的申论结构：先明确要讲的核心内容，再完成论证/反驳，最后总结内容对比赛的意义；
    - 先小结交锋，再推进立论，优先处理前序环节的核心攻防，再补充新的论证内容；
    - 驳论与申论结合，不能只反驳对方不强化己方，也不能只复述立论不处理交锋。

### （四）自由辩环节
1.  核心功能：集中攻防、放大己方优势、拆解对方核心战场，完成全场核心交锋的最终碰撞。
2.  执行规范：
    - 战场推进清晰，一个战场打完、完成作结后，再推进下一个战场，避免多个战场混乱交叉；
    - 团队配合默契，攻防分工明确，避免单人长时间发言、多人抢话；
    - 不回避核心问题，对方反复追问的核心质疑，必须给出明确回应，禁止全程逃问题。

### （五）总结陈词（结辩）环节
1.  核心功能：告诉评委「为什么我方赢了」，并让评委认同我方赢的方式，最终引导评委按照己方的思路述票。
2.  核心结构：反驳（为什么对方赢不了）+ 申论（为什么我方赢了）+ 升华（为什么我方赢得有意义）。
3.  执行规范：
    - 优先处理全场核心分歧，不纠结于零散的细节攻防，做到抓大放小；
    - 禁止全场复读机，不重复前场已经说过的内容，必须基于全场交锋完成新的梳理与总结；
    - 禁止纯煽情、不上价值不处理攻防，必须先完成核心交锋的比较，再进行价值升华；
    - 新攻防限制：仅可回应自由辩中新出现的攻防，禁止在结辩中提出前场从未出现的全新论点、全新论据。
4.  常用结辩结构模板：
    | 结构类型 | 核心逻辑 | 适用场景 |
    |----------|----------|----------|
    | 先驳后立 | 先拆解对方论证为什么不成立，再阐述我方论证为什么成立，最后价值升华 | 双方立论框架完全不同，几乎无重合核心分歧 |
    | 顺序型结构 | 按比赛时间线，梳理双方每个环节的交锋，最终论证每个核心战场都是我方占优 | 前场交锋细碎、混乱，小结未完成清晰的交锋梳理 |
    | 递进/递退结构 | 先说核心问题，退一步/进一步说次要问题，层层递进，最终完成整体论证 | 全场交锋集中，仅1-2个核心分歧，逻辑线较长 |
    | 归一结构 | 指出全场所有零散交锋，本质上都是同一个核心分歧，集中论证该核心分歧我方占优 | 场上交锋零散，但所有分歧都来自同一个核心矛盾 |

## 二、评委判准核心原则（Judge Agent执行判罚的核心依据）
1.  举证责任原则
    - 谁主张，谁举证：提出某个观点的一方，必须承担对应的举证责任；未完成举证的观点，不予采信。
    - 举证责任转移：一方完成初步举证后，举证责任转移给反对方；反对方未提出有效反驳的，该观点视为证成。
    - 常识豁免：符合大众普遍认知的常识性内容，无需额外举证；反常识的观点，提出方必须承担更高的举证责任。
2.  交锋采信原则
    - 双方达成共识的内容，直接采信，不再进行二次评判；禁止一方在后续环节推翻前场已确认的共识。
    - 针对同一个交锋点，一方完成有效论证，另一方未给出有效回应的，该战场判给完成论证的一方。
    - 双方均完成论证、交锋打平的，结合双方的举证效力、逻辑严谨性、贴合辩题的程度，综合判断；若仍无法区分，该战场双方均不得分。
3.  论证效力评判优先级
    - 完整逻辑链+权威实证+场景演绎的多层次论证，优先级远高于单一逻辑、纯理论推演的论证。
    - 有权威、可溯源来源的论据，优先级远高于无来源、个例、传闻类论据。
    - 贴合辩题现实背景、核心人群需求的论证，优先级远高于脱离现实的纯理论论证。
4.  违规内容判罚原则
    - 循环论证、稻草人谬误、偷换概念等逻辑谬误，未被对方指出的，评委可自行识别并不予采信；被对方明确指出且无法回应的，直接判定该论证不成立。
    - 编造数据、虚假论据、无来源论证，一经发现，直接否定该论点的论证效力，情节严重的，直接判定该方负向判罚。
    - 全程回避核心问题、不回应对方有效质疑的，对应核心论点直接判定为未证成。
5.  胜负最终判定标准
    - 核心判准：基于双方设定的、且达成共识的比较标准，哪一方的立场在该标准下完成了更充分的证成，哪一方获得胜利。
    - 核心维度：核心论点的证成度、核心交锋的胜负、价值框架的合理性、论证的严谨性与可信度。
    - 价值权重：在双方论证完成度相近的情况下，更贴合现实需求、更具备社会意义、更能给观众带来启发的一方，获得胜负倾向。
